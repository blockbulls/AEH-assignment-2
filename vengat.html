Script Description - TCP Port Scanner with Owner Information Retrieval using lua

Description: The script attempts to find the IP address and owner of an open TCP port, as well as the host status, other addresses, port status, and service on the target system.

Usage:

1. Install Nmap on your system.

2. Open a terminal or command prompt and navigate to the directory where the script is located.

3. Run the following command to scan a target system :

nmap --script <script-name> <target>

4. Replace <script-name> with the name of the script file (e.g. "vengat.nse") and <target> with the IP address or hostname of the target system.

5. The script will scan for open TCP ports on the target system and query the identd daemon on port 113 to determine the owner of each open port. If DNS scanning is enabled, it will also perform reverse DNS lookups on each IP address found to retrieve the corresponding hostnames.
